# Contributing

Contributions are welcome.

Before opening a pull request:

- Keep upstream copyright and AGPL notices intact.
- Do not commit `.env` files, logs, databases, generated binaries, or private
  deployment files.
- Include focused tests for behavior changes when practical.
- Rebuild frontend assets when changes under `web/src` need to be served by the
  embedded Go backend.
